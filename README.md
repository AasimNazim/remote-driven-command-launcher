# Remote Driven Command Launcher

A Python-based remote automation system that allows users to control a computer using SMS commands received through a mobile device.

The system interprets incoming messages and performs predefined computer actions.

## Features

- Remote command execution
- SMS-based communication
- Open Microsoft applications
- Shutdown system remotely
- Execute automated tasks

## Technologies

- Python
- SMS Communication
- Automation
- ADB Tools
- Windows System Control
